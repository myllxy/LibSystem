# LibSystem
:recycle: 简介：基于Swing的图书管理系统  

主要实现功能：  

1.普通用户，管理员登陆注册  
2.普通用户查阅图书并借阅，管理员管理借阅信息及书库信息  
3.普通用户，管理员可修改账户密码
